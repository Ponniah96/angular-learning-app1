Angular Templates

2.Templates

Templates contains HTML code of the component where we can either add using template method in Component itself or we can create seperate HTML file and also by calling using templateUrl.

import { Component } from '@angular/core';
@Component {(
template: 'helloworld.html'
)}
export class HelloWorldComponent {
message:'hello-world'
}
