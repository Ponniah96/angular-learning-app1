Angular Components

1.Components

Components are the building blocks of the angular application.

Like React Components, angular components contains which is reusable.

Angular Components contains @Component decorator which contains basic properties such as:

  • CSS Selector: Value we are using here is the actual name of the component which we can reuse as html tag in some other components 
    @Component {(
selector: 'hello-world',
)} 

In another component HTML,
<hello-world> </hello-world>
  • TemplateUrl: It will maps HTML template for this component
  • styleurl: It will add styles to this component.

Example:

import { Component } from '@angular/core';
@Component {(
selector: 'hello-world',
template: '<h1> Hi</h1>'
)}
export class HelloWorldComponent {
// The code in this class drives the component's behavior.
}
