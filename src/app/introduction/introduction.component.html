Angular Basics

Introduction:

Angular is application designed framework and development platform which is mainly used to design single page sophisticated apps

Click here to know more about angular from official website.

Features of Angular:

  • Angular is a component based framework
  • Angular has its own rich in-build libraries like routing, forms,Http client services and so on
  • Angular has its own CLI (command line interface) to develop, build, test application

Pre Requisites of Angular:
  • HTML
  • CSS
  • JavaScript (Classes)
  • TypeScript

Core Functions of Angular:

1.Components

Components are the building blocks of the angular application.

Like React Components, angular components contains which is reusable.

Angular Components contains @Component decorator which contains basic properties such as:

  • CSS Selector: Value we are using here is the actual name of the component which we can reuse as html tag in some other components 
        @Component {(
    selector: 'hello-world',
    )} 
    
    In another component HTML,
    <hello-world> </hello-world>
  • TemplateUrl: It will maps HTML template for this component
  • styleurl: It will add styles to this component.

Example:

    import { Component } from '@angular/core';
    @Component {(
    selector: 'hello-world',
    template: '<h1> Hi</h1>'
    )}
    export class HelloWorldComponent {
    // The code in this class drives the component's behavior.
    }

2.Templates

Templates contains HTML code of the component where we can either add using template method in Component itself or we can create seperate HTML file and also by calling using templateUrl.

import { Component } from '@angular/core';
@Component {(
template: 'helloworld.html'
)}
export class HelloWorldComponent {
    message:'hello-world'

3.Dependency Injection

Dependency Injection helps you to declare dependency of your angular components.

Angular creates own instantiation with these dependency.

Example: Click on below button and check console

In service.ts 

import { Injectable } from '@angular/core';
@Injectable ({
    providedIn: 'root'
})
export class IntroductionServiceService {
    constructor ()  { }
    increaseCountValue(count:number){
    console.log('Dependency Injection Count value: ',count+1);
    }
}    

In Component.ts 
import { IntroductionServiceService } from '../introduction-service.service';
export class IntroductionComponent {
    count=0;
    constructor(private dependencyInjectionServiceComponent:IntroductionServiceService){}
    increaseCount(){
    this.dependencyInjectionServiceComponent.increaseCountValue(this.count);
    this.count++;
    }
}

Angular CLI:

Angular CLI is the command prompt like NPM to install, test, build angular application

Important Commands for angular CLI:

Install angular CLI:
npm install -g @angular/cli
ng new my-app
cd ./my-app 

Get local machine access:
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Run Local app:
ng serve --open

Create Component:
ng generate component xyz

Test application:
ng test

Add Angular Material UI:
ng add @angular/material

Add PWA in our app:
ng add @angular/pwa

Build your application:
ng build

Create Service for our application:
ng generate service introduction-service

Deploy Application using GitHub:
ng build --output-path docs --base-href my-app
git add . 
git status 
git commit -m "Push Code Comments"
git push -u origin main

Router

Angular router is used to navigate angular app from single page to multiple apps

Example:

Navigation Component 1 | Navigation Component 2 
        
In app-routing-module.ts 
import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';

import { NavigationComponentComponent } from './navigation-component/navigation-component.component';
import { Navigation2Component } from './navigation2/navigation2.component';

const routes: Routes = [
{path:'nav1', component:NavigationComponentComponent},
{path:'nav2', component:Navigation2Component}
];

@NgModule({
    imports: [RouterModule.forRoot(routes)],
    exports: [RouterModule]
})
export class AppRoutingModule { }

In introduction.component.html
<a class="link" [routerLink]="['/nav1']">
    Navigation Component 1
</a>
<a class="link" [routerLink]="['/nav2']">
    Navigation Component 2
</a>

Forms

Angular Forms are the build in forms which is available in angular library to build simple as well as complex forms.

        
In app.module.ts
import { ReactiveFormsModule } from '@angular/forms';
@NgModule({
    imports: [
        ReactiveFormsModule
    ],
})

In introduction.component.ts
import { FormBuilder } from '@angular/forms';
export class IntroductionComponent {
    basicForm=this.formBuilder.group({
        name:'',
        age:0,
        gender:'',
        role:''
        });  
    constructor(
        private formBuilder:FormBuilder,
        ){}  
    onSubmit(): void {
        console.log('Form values: ',this.basicForm.value);
        this.basicForm.reset();
        }
    }

In introduction.component.html
<form [formGroup]="basicForm" (ngSubmit)="onSubmit()">
    <div>
        <label for="name">Name</label>
        <input id="name" placeholder="Enter Name"  type="text" formControlName="name">
    </div>
    <button type="submit" class="button">Submit</button>
</form>
